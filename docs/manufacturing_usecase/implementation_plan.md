# 「製造業向け 工程管理」ユースケースの追加計画

## 背景・目的
ご提示いただいたExcelの図表（製造計画・受注データベース・各工程の進行管理）をHTML/CSSで再現し、「つなげモン モバイル」の新たなユースケースとして追加します。

既存の「軽貨物管理システム（物流）」のダイアグラムと同様に、Tailwind CSSとLucideアイコンを活用し、視覚的にわかりやすくリッチな図解を構築します。

## Open Questions

> [!IMPORTANT]
> この新しい図表（製造業向け）をサイトのどこに配置するか、ご希望の構成をお選びください。

**案1: 既存の詳細ページ（`usecase_mobile.html`）の「下部（2つ目のセクション）」に追加する**
- メリット: 「つなげモン モバイル」のユースケース（物流・製造）が1ページにまとまるため、ユーザーがスクロールするだけで複数の事例を見ることができます。

**案2: 新しい独立ページ（例: `usecase_factory.html`）として作成する**
- メリット: 今後さらに別の事例が増えた場合でも、URLが分かれているため個別にリンクを送りやすくなります。（LPのTheme 05あたりからリンクを貼る想定）

**案3: （LP用）LP内の「Theme 05: つなげモン IoT」の図表として組み込む**
- モバイルアプリ（タブレット）というより、製造業やIoTの文脈が強いため、LPの該当箇所に直接組み込むパターンです。

## Proposed Changes

（※案1または案2の独立セクションとして作る場合の実装イメージ）

### 1. 図解の構造（HTML/CSS）
- **上部**: 「製造計画」「受注登録（他システムとの連携）」ブロックと、そこから流れる矢印。
- **中央**: 「受注データベース」のアイコン。ここから下方向および横方向にデータが分岐・展開する矢印を配置。
- **横並びブロック**: 「工程1 ➡ 工程2 ➡ 工程3 ➡ 工程n」という進捗を表すフロー図。
- **下部**: 今回作成したスマートフォン/タブレットのCSSフレーム（`.phone-frame`）を使い、画面内に「製造ロット xxx」「工程2」「完了」ボタンを配置。
- **吹き出し**: 「タブレット・スマートフォンからタッチのみで進行登録が可能」というテキストを、Tailwindの装飾を使ってスマホの横に吹き出し風に配置します。

### 2. デザインのトーン＆マナー
- 既存のダイアグラム（青系の矢印、ネイビーのテキスト、白いカード背景）を踏襲し、サイト全体のトンマナに合わせます。
